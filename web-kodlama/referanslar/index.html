Açık Sayfalar – Web Dersleri Sözlüğü

Bu tabloda web derslerinde kullanılan HTML/CSS/JS kavramları, açıklamaları, örnek kodları ve kullanım alanları listelenmektedir.

Keyword / Tag Açıklama Örnek Kod Kategori
<!-- ... --> HTML yorumlarını tanımlar, tarayıcı tarafından görünmez. 
<!-- Bu bir yorumdur -->
 HTML / Yorum
<!DOCTYPE> Belgenin türünü tanımlar ve tarayıcıya HTML sürümünü bildirir. 
<!DOCTYPE html>
 HTML / Belge Yapısı
<a> Bir bağlantı (link) tanımlar; kullanıcıyı başka bir sayfaya veya siteye yönlendirir. 
<a href="https://example.com">Example</a>
 HTML / Bağlantı
<abbr> Kısaltma veya akronim tanımlar; üzerine gelindiğinde tam açılımı gösterilebilir. 
<abbr title="HyperText Markup Language">HTML</abbr>
 HTML / Metin
<acronym> HTML5’te desteklenmez. Kısaltma veya akronim tanımlamak için <abbr> kullanın. 
<acronym title="World Health Organization">WHO</acronym>
 HTML / Metin
<address> Belgenin yazarına veya sahibine ait iletişim bilgilerini tanımlar. 
<address>
Okan Kaplan

info@example.com

www.example.com
</address>
 HTML / İletişim
<applet> HTML5’te desteklenmez. Gömülü applet için <embed> veya <object> kullanın. 
<applet code="MyApplet.class" width="300" height="300"></applet>
 HTML / Gömülü Uygulama
<area> Bir resim haritası (image map) içindeki tıklanabilir alanı tanımlar. 
<img src="example.jpg" usemap="#map1">
<map name="map1">
  <area shape="rect" coords="34,44,270,350" href="page.html" alt="Link">
</map>
 HTML / Resim Haritası
<article> Bağımsız, kendi başına anlam taşıyan bir içerik bölümünü tanımlar. 
<article>
  <h2>Makale Başlığı</h2>
  <p>Makale içeriği buraya gelir.</p>
</article>
 HTML / Yapı
<aside> Sayfa içeriğinden bağımsız, tamamlayıcı veya yan bilgi bölümü tanımlar (ör. kenar çubuğu, ek notlar). 
<aside>
  <h3>Ek Bilgi</h3>
  <p>Bu, ana içerikten bağımsız bir yan not örneğidir.</p>
</aside>
 HTML / Yapı
<audio> Sayfaya gömülü ses içeriği ekler ve kullanıcıya oynatma kontrolleri sağlar. 
<audio controls>
  <source src="ses.mp3" type="audio/mpeg">
  Tarayıcınız audio etiketini desteklemiyor.
</audio>
 HTML / Medya
<base> Belgedeki tüm göreli (relative) URL’ler için temel URL ve hedefi belirler. 
<head>
  <base href="https://example.com/" target="_blank">
</head>
 HTML / Başlık
<bdi> Dışındaki metin yönünden bağımsız olarak bir metin parçasını izole eder (metin yönü farklı olabilir). 
<p>Normal metin <bdi>متن عربی</bdi> devam ediyor.</p>
 HTML / Metin Yönü
<bdo> Mevcut metin yönünü geçersiz kılar ve belirlenen yönde metin gösterir. 
<p>Normal metin <bdo dir="rtl">Sağdan sola yazı</bdo> devam ediyor.</p>
 HTML / Metin Yönü
<blockquote> Başka bir kaynaktan alınmış alıntı içeren bir bölümü tanımlar. 
<blockquote cite="https://example.com">
  Bu, başka bir kaynaktan alınmış alıntıdır.
</blockquote>
 HTML / Metin
<body> Belgenin ana içeriğini tanımlar; tüm görünür içerikler bu etiketin içinde yer alır. 
<body>
  <h1>Merhaba Dünya</h1>
  <p>Bu, sayfanın ana içeriğidir.</p>
</body>
 HTML / Belge Yapısı
<br> Tek satırlık bir boşluk (satır sonu) ekler. 
<p>Satır 1<br>Satır 2</p>
 HTML / Metin
<button> Tıklanabilir bir düğme (buton) tanımlar. 
<button type="button">Tıkla Beni</button>
 HTML / Form Kontrolü
<canvas> Grafikleri anlık olarak çizmek için kullanılır (genellikle JavaScript ile). 
<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  var c = document.getElementById("myCanvas");
  var ctx = c.getContext("2d");
  ctx.fillStyle = "red";
  ctx.fillRect(20, 20, 150, 50);
</script>
 HTML / Grafik
<caption> Bir tablonun başlığını tanımlar ve tablo üzerinde görünür. 
<table border="1">
  <caption>Örnek Tablo Başlığı</caption>
  <tr><th>>Ad</th><th>Yaş</th></tr>
  <tr><td>Ali</td><td>25</td></tr>
</table>
 HTML / Tablo
<cite> Bir eserin, kitabın, makalenin veya çalışmanın başlığını tanımlar. 
<p>Bu fikir <cite>Web Geliştirme Rehberi</cite> kitabından alınmıştır.</p>
 HTML / Metin
<code> Bilgisayar kodu parçasını tanımlar ve genellikle monospace fontla gösterir. 
<p>JavaScript örneği: <code>alert('Merhaba Dünya');</code></p>
 HTML / Kod
<col> <colgroup> içindeki her sütun için stil ve özellikleri belirler. 
<table border="1">
  <colgroup>
    <col span="1" style="background-color:lightblue">
    <col span="1" style="background-color:lightgreen">
  </colgroup>
  <tr><td>A</td><td>B</td></tr>
</table>
 HTML / Tablo
<colgroup> Tablodaki bir veya birden fazla sütun için toplu stil veya özellikleri tanımlar. 
<table border="1">
  <colgroup>
    <col span="2" style="background-color:lightyellow">
  </colgroup>
  <tr><td>A</td><td>B</td></tr>
</table>
 HTML / Tablo
<data> Bir içeriğin makine tarafından okunabilir değerini ekler. 
<p>Yaşım: <data value="30">30 yaş</data></p>
 HTML / Veri
<datalist> Bir giriş (input) alanı için önceden tanımlanmış seçenekler listesi sağlar. 
<input list="renkler" name="renk">
<datalist id="renkler">
  <option value="Kırmızı">
  <option value="Mavi">
  <option value="Yeşil">
</datalist>
 HTML / Form Kontrolü
<dd> Açıklama listesinde bir terimin değerini veya açıklamasını tanımlar. 
<dl>
  <dt>HTML</dt>
  <dd>Web sayfalarını oluşturmak için kullanılan işaretleme dili.</dd>
</dl>
 HTML / Liste
<del> Belgeden silinmiş metni tanımlar ve genellikle üstü çizili olarak gösterir. 
<p>Bu cümlede <del>eski</del> yeni kelime kullanılmıştır.</p>
 HTML / Metin
<details> Kullanıcının görüntüleyip gizleyebileceği ek bilgileri tanımlar. 
<details>
  <summary>Daha fazla bilgi</summary>
  Bu içerik kullanıcı tarafından açılıp kapatılabilir.
</details>
 HTML / Etkileşim
<dfn> İçerik içinde tanımlanacak bir terimi belirtir. 
<p>HTML5 dilinde <dfn>section</dfn> etiketi, belgede bağımsız bir bölüm oluşturur.</p>
 HTML / Metin
<dialog> Bir iletişim kutusu veya pencere tanımlar ve kullanıcı etkileşimi için açılıp kapanabilir. 
<dialog id="myDialog">
  Bu bir dialog kutusudur.
  <button onclick="document.getElementById('myDialog').close()">Kapat</button>
</dialog>

<button onclick="document.getElementById('myDialog').showModal()">Dialogu Aç</button>
 HTML / Etkileşim
<div> Belge içinde bir bölüm veya konteyner tanımlar, içerik gruplamak için kullanılır. 
<div style="background-color:lightgray; padding:10px;">
  <p>Bu bir div içinde yer alan paragraftır.</p>
</div>
 HTML / Bölüm
<dl> Açıklama listesi (description list) tanımlar ve terim-değer çiftlerini içerir. 
<dl>
  <dt>HTML</dt>
  <dd>Web sayfalarını oluşturmak için kullanılan işaretleme dili.</dd>
  <dt>CSS</dt>
  <dd>Web sayfalarının stilini tanımlayan dil.</dd>
</dl>
 HTML / Liste
<dt> Açıklama listesinde (description list) bir terim veya adı tanımlar. 
<dl>
  <dt>HTML</dt>
  <dd>Web sayfalarını oluşturmak için kullanılan işaretleme dili.</dd>
</dl>
 HTML / Liste
<em> Vurgulanmış (emphasized) metni tanımlar ve genellikle italik olarak gösterilir. 
<p>Bu bir <em>önemli</em> paragraftır.</p>
 HTML / Metin
<embed> Dış bir uygulama veya multimedya içeriği gömmek için bir konteyner tanımlar. 
<embed src="video.mp4" width="400" height="300" type="video/mp4">
 HTML / Medya
<fieldset> Formdaki ilgili alanları gruplamak için kullanılır ve genellikle bir başlık () ile birlikte gösterilir. 
<form>
  <fieldset>
    <legend>Kişisel Bilgiler</legend>
    <label>Ad: </label><input type="text"><br>
    <label>Soyad: </label><input type="text">
  </fieldset>
</form>
 HTML / Form
<figcaption> <figure> elemanı için bir başlık veya açıklama tanımlar. 
<figure>
  <img src="resim.jpg" alt="Örnek Resim">
  <figcaption>Bu resim bir örnek görseldir.</figcaption>
</figure>
 HTML / Görsel
<figure> Kendi başına anlam taşıyan içerik (ör. resim, çizim, grafik) tanımlar ve genellikle bir başlık ( 
<figure>
  <img src="resim.jpg" alt="Örnek Resim">
  <figcaption>Bu resim bir örnek görseldir.</figcaption>
</figure>
 HTML / Görsel
<footer> Belge veya bölüm için alt bilgi (footer) tanımlar. Genellikle iletişim bilgisi, telif hakkı veya navigasyon linkleri içerir. 
<footer>
  <p>© 2026 Örnek Şirket. Tüm hakları saklıdır.</p>
</footer>
 HTML / Bölüm
<form> Kullanıcıdan veri girişi almak için bir HTML formu tanımlar. 
<form action="/submit" method="post">
  <label>Ad: </label><input type="text" name="ad"><br>
  <label>E-posta: </label><input type="email" name="email"><br>
  <button type="submit">Gönder</button>
</form>
 HTML / Form
<h1> … <h6> HTML başlıklarını tanımlar. <h1> en önemli başlık, <h6> en az önemli başlıktır. 
<h1>Ana Başlık</h1>
<h2>Alt Başlık 1</h2>
<h3>Alt Başlık 2</h3>
 HTML / Başlık
